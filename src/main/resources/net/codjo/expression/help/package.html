Ce package regroupe les classes spécialisées dans la documentation de {@link com.agf.expression.FunctionHolder}.
Utilisation du package
Il est préférable de passer par le {@link com.agf.expression.FunctionManager#getAllFunctionsHelp()} pour avoir l'aide (utilisation du comportement par défaut).

Extension

Le point d'extension de ce framework est symbolisé par l'interface FunctionHolderHelp. Pour définir une aide spécifique pour un FunctionHolder particulier il faut lui faire implanter l'interface FunctionHolderHelp (exemple {@link com.agf.expression.KernelFunctionHolder} ).

NB Si un FunctionHolder implémente l'interface FunctionHolderHelp alors il sera utilisé à la place du comportement par défaut.
Résumé des classes
  • {@link com.agf.expression.help.FunctionHolderHelp} : Interface principale
  • {@link com.agf.expression.help.DefaultFunctionHolderHelp} : Aide par défaut.
  • {@link com.agf.expression.help.UserFunctionHolderHelp} : Aide par défaut pour les UserFunctionHolder.
